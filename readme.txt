# wp-show-posted-Numbers

Overview
## Description
wp-show-posted-Numbers is wordpress plugin which shows your wordpress post history.

- Total Posts
- Total days
- Post days
- No post days
- First Post Date
- Last Post date
- Daily posts(Graph)
- By Weekdays(Graph)
- Total Posts(Graph)
